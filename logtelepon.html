Nama : Log Telepon (Magang Diskominfo Bantul)

Tools : Laravel, Booststrap

Deskripsi

LogTelepon adalah aplikasi hasil projek tim 4 orang, yang merupakan arahan dari mentor Magang saat di Dinas Komunikasi dan Informatika Kabupaten Bantul. Tujuan dibuatnya aplikasi ini adalah untuk mencatat panggilan masuk dan panggilan keluar serte keterangan penting secara manual.

Pembuatan aplikasi ini memanfaatkan figma untuk membuat desain UI/UX, serta pertama kalinya saya dan tim pengembang lainnya mengenal framework laravel. Pada aplikasi ini laravel digunakan untuk mengembangkan back end fungsi CRUD, lalu di integrasikan ke tampilan utama dalam bentuk tabel sederhana. Meski saat itu tim belum cukup menguasai framework laravel, tapi arahan dari mentor cukup membantu untuk saya dan tim belajar hal baru.
