Nama : YokNgathering

Tools : Codeigniter, Booststrap

Deskripsi

YokNgathering adalah aplikasi hasil dari projek tim 3 orang dengan peran masing-masing yaitu, data analyst, back end, dan saya sendiri sebagai front end. Untuk mengikuti event Fipex#4 pameran karya perangkat lunak mahasiswa UAD, tim kami mengambil tema pemesanan cathering. Tujuan dibuatnya aplikasi ini untuk memberikan ide bagaimana jika jasa cathering bisa dipesan secara online melalui aplikasi berbasis web.

Dalam pengembangan aplikasi ini, saya dan tim mengawalinya dengan melakukan pengumpulan data melalui wawancara langsung dengan salah satu pelaku usaha cathering di Ledoksari, Wonosari, Gunungkidul, dengan tujuan data wawancara tersebut bisa menjadi dasar bagi data analyst untuk memperkirakan kebutuhan data pengguna, keperluan fitur pada desain mock up, hingga menjadi prototype sebelum dikembangan dalam bentuk source code. Untuk pembuatan desain saya menggunakan figma, setelah desain sudah jadi, masuklah ke tahap pengembangan kode program aplikasi dan inilah pertama kali saya kenal dengan framework codeigniter karena saran dari dosen.
