Nama : APPSIPP Kepek II (WebGIS)

Tools : Laravel, Bootstrap, Leaflet, Breeze, Spatie

Deskripsi

APPSIPP Kepek II merupakan karya saya sebagai fullstack developer berdasarkan tema skripsi yang saya ambil (Sistem Informasi Pemetaan Penduduk Kepek II Berbasis WebGIS dengan Pendekatan Metode Waterfall) selama saya menyelesaikan pendidikan sebagai mahasiswa Universitas Ahmad Dahlan jurusan Sistem Informasi. Fokus utama pada aplikasi ini adalah memetakan data Penduduk berdasarkan titik koordinat longitude dan latitude pada peta. Adapun fitur lainnya, seperti mencatat data pergerakan penduduk mulai dari imigrasi, emigrasi, natalitas, hingga mortalitas penduduk.

Pembuatan aplikasi ini menggunakan framework utama Laravel sebagai back end, mulai dari pengembangan konsep MVC dari pemanfaatan route,model, dan controller hingga fitur CRUD dapat di implementasikan di tampilan UI. Sedangkan untuk tampilan antarmuka saya menggunakan bootstrap karena mempunyai dokumentasi lengkap dan mudah digunakan, serta pemanfaatan leaflet js sebagai alat untuk membuat peta interaktif yang dibagi 3 jenis, yaitu streetmap, citra satelit, dan peta kombinasi. Fitur keamanan juga dipertimbangkan, karena aplikasi yang Saya kembangkan memuat data sensitif yang terdapat pada Kartu Keluarga seperti nomor KK, NIK, alamat dll, maka diperlukan fitur keamanan untuk menjaga kumpulan data tersebut. Oleh karena itu saya memanfaatkan laravel breeze untuk melakukan instalasi lengkap fitur login, yang di kombinasikan dengan Spatie untuk membagi level user menjadi tiga, yaitu Super Admin dengan keuntungan punya semua hak akses, admin dan pengguna umum dengan batasan akses tertentu.

Selama saya membuat aplikasi yang memakan banyak waktu ini, saya belajar banyak hal, salah satunya ya tentu saja "Hidup tidak bisa serba instan, semua butuh usaha", bagaimana tidak? aplikasi ini tidak bisa langsung jadi begitu saja. Ceritanya setelah selesai lolos seminar proposal, saya tidak langsung melanjutkan menulis bab 4 dan 5 skripsi bahkan tidak langsung membuat kode program aplikasi ini, saya mogok 3 bulan dari penelirian yang saya dalami. Pada waktu 3 bulan itu saya berhenti dari penelitian skripsi bukan karena tidak mau melanjutkan/malas, melainkan saya menggunakan waktu 3 bulan itu untuk menguasai PHP dasar, masih harus belajar relasi basis data seperti one to one, one to many, dan many to many dan bagaimana cara menerapkannya di framework laravel. Leaflet js pun juga baru pertama kali, saya sampai harus belajar tentang teknologi peta, bagaimana cara install leaflet dan menggunakannya di laravel, serta bagaimana cara melakukan looping database pada peta leaflet. Setelah semua itu saya pelajari, saya langsung membuat aplikasi ini mulai dari merancang database, cara kerja, dan tampilan antarmuka dalam bentuk source code. Begitu fungsi utama aplikasi sudah jadi, tahap testing dilakukan untuk memastikan tidak ada bug atau error pada fungsi utama, mulai dari test kesuksesan CRUD sampai test login dan register.

Dari semua pengalaman yang saya alami selama membuat aplikasi, menurut saya ilmu yang saya miliki saat ini masih sangat jauh. Tapi itu sudah cukup membuktikan kalau saya juga siap untuk update ilmu baru jika dibutuhkan. Saya bersyukur karena sampai hari ini ilmu web development yang saya pelajari masih bisa digunakan.
